Il file contiene  i dati geografici (in formato shapefile) riguardanti le Province relativamente al suo anno di riferimento. I dati sono disponibili nella proiezione geografica WGS84 UTM Zona 32N. 

I dati contenuti nel file sono i seguenti:

1. COD_PROV: codice della Provincia. 

2. DEN_PROV: denominazione della Provincia.
